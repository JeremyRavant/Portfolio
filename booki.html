Objectifs pédagogiques

Intégrer du contenu conformément à une maquette avec HTML et CSS

Contexte

Pour ce projet, je suis développeur web au sein de la start-up Booki, qui propose des hébergements et activités dans plusieurs villes. Sarah, la CTO de Booki, a fait appel à mes services car elle voudrait permettre aux usagers de trouver des hébergements et des activités dans la ville de leur choix. Pour ce faire, je suis missionné d’intégrer l’interface du site à partir de la maquette visuelle fournie par Loïc, l’UI Designer. J’ai comme contrainte de n’utiliser que du HTML et CSS pour l’intégration du site.

Problèmatiques rencontrées

Ici, la problématique majeur rencontré dès le départ était de ne pas avoir une structure HTML désorganisée. J'ai donc structuré ce code avec des balises sémantiques. On a ensuite nommé clairement les classes CSS pour faciliter notre mise en forme. La clé pour ce projet a été de travailler par étape, ne pas tout faire en même temps. J'ai été bien aidé grâce à la maquette Figma et à énormément de ressources trouvées partout sur internet.

Compétences acquises

  1. Utilisation de VS Code
  2. Importation de polices depuis Google Fonts
  3. Versioning avec Git et GitHub
  4. Gestion du positionnement des éléments
  5. Gestion de l’espacement entre les éléments
  6. Utilisation de Flexbox
  7. Création d’un formulaire
  8. Gestion des breakpoints
  9. Validation du HTML et CSS via le W3C
